# Voyagerpad

Voyagerpad is a 7 key macropad with an OLED Display.

## Features:
- 128x32 OLED Display
- 7 Keys

## PCB
The PCB was made in KiCad.

### BOM
Here is everything needed to make this macropad:
- 1x XIAO RP2040
- 1x 0.91" 128x32 OLED Display
- - 7x Cherry MX Switches
- 7x DSA Keycaps
- 1x Case
